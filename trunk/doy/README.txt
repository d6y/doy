About Unit Tests
================

Uses http://openjsan.org/doc/t/th/theory/Test/Simple/0.21/index.html

Load tests/all-tests.html into Safari to run all the tests.  This fetches an example HTML page (tests/listing.html) via the code.google.com web site.

dump-listings.html is a utility/sanity check to parse the contents of listing.html and show them in the HTML page.
